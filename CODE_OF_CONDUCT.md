# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

We want this to be a welcoming, respectful project for everyone, regardless of
background or experience. Be kind, assume good faith, and focus on what is best
for the community.

## Scope

This applies in all project spaces — issues, pull requests, discussions — and
when representing the project in public.

## Reporting

To report a concern, contact the maintainer at **info@effizienzagent.de**. Reports
are handled confidentially. Maintainers will review and respond as outlined in
the Contributor Covenant's enforcement guidelines.
